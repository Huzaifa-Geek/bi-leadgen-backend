#  Business Intelligence & Lead Generation Platform

Production-style FastAPI backend for scraping, scoring, and managing business leads.

##  Features

- JWT Authentication
- Role-based access (User/Admin)
- Background scraping with Celery + Redis
- Job status tracking
- Lead scoring engine
- Google Maps scraping via SerpAPI
- CSV export
- Rate limiting
- Multi-user data isolation

---

##  Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL / MySQL
- Celery
- Redis
- SerpAPI
- Pydantic
- JWT
